# Global Game Jam 2021
## "Lost and Found"
## Games: Senses

### Evolution de l'idée:

- 1 - Créer un niveau ou on doit chercher une épée dans un scroller
- 2 - Créer une foret ou le joueur se perds et doit retrouver une carte qui lui indique la sortie
- 3 - Créer une labyrinte vertical avec plusieurs couloir
- 4 - Créer une labyrinte vertical avec plusieurs couloir ou on passerai de coulois en coulois avec un portal gun
- 5 - Créer une serie de niveau avec des jeu de perpective genre un niveau comme la réalité mais avec un ciel rose et des flamme bleu
- 5.1 - Créer un niveau où le joueur est perdu dans une salle au mirroir
- 5.2 - Un niveau medival fantaisie
- 5.3 - un portal gun pour sortir du niveau
- 5.4 - Ajouter un jumper et un saut de l'ange
- 5.5 - Combat avec un portal gun et une épées

### Competence Apprise durant cette gamejam:

- L'uttilisation des réflection
- Retarget une animation
- Load un level avec du code C++

Developed with Unreal Engine 4
